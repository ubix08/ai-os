# Content Templates

## Key Takeaways Box
```markdown
> **Key Takeaways**
> - Main point one with specific detail
> - Main point two with specific detail
> - Main point three with specific detail
```

## FAQ Section
```markdown
## FAQ

### How do I [common task]?
[Direct answer in 1-2 sentences. Include keyword naturally.]

### What is the difference between X and Y?
[Concise comparison. 1-2 sentences.]
```

## Definition Block
```markdown
**Term**: Concise definition (1-2 sentences). Include context about when and why to use it.
```

## Author Bio
```markdown
**Author Name** is a [role] with [X years] of experience in [domain]. 
They specialize in [area]. [Optional: link to portfolio/GitHub/twitter.]
```

## Code Blocks
```markdown
```python
# python
import library

# Explanation of what this does
result = library.do_thing()
print(result)
```
```

## Comparison Table
```markdown
| Feature | Option A | Option B |
|---------|----------|----------|
| Feature 1 | ✅ | ❌ |
| Feature 2 | ✅ | ✅ |
| Price | Free | $29/mo |
| Best for | Small projects | Enterprise |
```

## Step-by-Step Instructions
```markdown
1. **First step**: Brief instruction with context
   ```bash
   command to run
   ```
2. **Second step**: Expected output or result
   Continue with detailed guidance.
```

## Internal Link
```markdown
For more details, see our guide on [Related Topic](/articles/YYYY-MM-DD-related-topic).
```

## Warning / Callout
```markdown
> **⚠️ Warning**: Important caveat or common pitfall to avoid.
> **💡 Tip**: Helpful shortcut or best practice.
```

## Frontmatter Template
```markdown
---
title: "Title: Subtitle if Needed"
description: "Meta description 150-160 chars with target keyword"
date: YYYY-MM-DD
lastUpdated: YYYY-MM-DD
tags: [primary-topic, secondary-topic, tutorial]
author: "Full Name"
---
```
