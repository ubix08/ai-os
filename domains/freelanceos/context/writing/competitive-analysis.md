# Competitive Analysis Framework

## Research Pipeline

### Phase 1: Discovery
1. Search for the target keyword and related terms
2. Identify top 5-8 ranking articles
3. Record titles, authors, publication dates, word counts

### Phase 2: Deep Analysis
Score each competitor on 6 dimensions (0-10):

| Dimension | Description |
|-----------|------------|
| Content Depth | Coverage completeness, word count, detail level |
| Technical Accuracy | Correctness of facts, code, and procedures |
| SEO Structure | Heading hierarchy, keyword placement, internal links |
| AI Citation Readiness | Key Takeaways, FAQ, structured data, JSON-LD |
| E-E-A-T Signals | Author credentials, sourced stats, date freshness |
| Uniqueness | Information gain — what does it add beyond the obvious |

### Phase 3: Gap Analysis
Identify gaps per competitor:
- **Content gap**: Missing subtopics or sections
- **Depth gap**: Surface-level treatment of important points
- **Angle gap**: Different framing or perspective
- **Format gap**: Missing tables, diagrams, code, FAQ
- **Freshness gap**: Outdated information or versions
- **Trust gap**: Weak author credentials, no sources

### Phase 4: Synthesis
Produce a killer blueprint with:
- **Must-haves**: Elements every top article includes
- **Differentiation opportunities**: Gaps to exploit
- **Information gain**: What unique value we'll provide
- **Target structure**: Outline with sections and subsections
- **Sourcing plan**: Key statistics and references to gather

## Research Report Output
Save to `content/research/YYYY-MM-DD-topic-research.md`
Include: executive summary, competitor scores table, gap analysis, killer blueprint, sources.
